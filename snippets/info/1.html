The "ReminderNotes.js" file for the Custom Widget.

We use AMD when creating a new custom widget.

For 5.14.1 and lower we use 'require' to load our custom widget.

For 5.15 and higher we must use 'define' to load our custom widget.

We 'declare' a new AMD module with a name.

We pass through the JavaScript files as functions inside the declared AMD module.
